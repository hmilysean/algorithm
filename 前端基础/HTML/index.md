# HTML是什么

HTML 是用来描述网页的一种语言。
* HTML 指的是超文本标记语言: HyperText Markup Language
* HTML 不是一种编程语言，而是一种标记语言,有一套标记标签(markup tag)。
* HTML 文档包含了HTML标签及文本内容 
* HTML 使用标记标签来描述网页，HTML文档也叫做web页面。
